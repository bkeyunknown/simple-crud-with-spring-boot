# Simple CRUD with Spring Boot
## Database
- Mahasiswa
- Fakultas
- Jurusan
- Mata Kuliah

# How to run
1. Clone repository ```git clone https://github.com/bkeyunknown/spring_data_mahasiswa.git```
2. Start Spring Boot ```mvn spring-boot:run```
3. Enjoy
